Todo lo que necesitas

Diseñado para equipos que gestionan múltiples cuentas AWS y clusters EKS a escala.

🔍

Auto-detección de profiles

Lee automáticamente todos los profiles de tu ~/.aws/config. Sin configuración adicional.

⚡

Procesamiento paralelo

Usa ThreadPoolExecutor con workers configurables para procesar cientos de profiles en minutos.

💾

Backup automático

Crea un respaldo de tu kubeconfig existente antes de sobrescribirlo. Siempre seguro.

🔒

Autenticación dinámica

Usa aws eks get-token para tokens renovables. Sin credenciales estáticas en el kubeconfig.

🎯

Filtrado por profiles

Procesa todos los profiles o selecciona solo los que necesitas con --profiles.

🧪

Modo dry-run

Previsualiza el kubeconfig que se generaría sin modificar nada en tu sistema.

Cómo funciona

Tres pasos. Sin complicaciones.

1

Escanea tus profiles

Lee ~/.aws/config y detecta todos los profiles AWS configurados, incluyendo SSO.

2

Descubre clusters

Consulta la API de EKS en paralelo para cada profile y región, listando todos los clusters disponibles.

3

Genera el kubeconfig

Crea un archivo kubeconfig válido con clusters, usuarios, contextos y autenticación dinámica.

Instalación rápida

Listo en menos de un minuto.

# Clonar el repositorio
git clone https://github.com/yourusername/auto-kubeconfig.git
cd auto-kubeconfig

# Instalar dependencias
pip install -r requirements.txt

# Ejecutar
python auto_kubeconfig.py

# O instalar como paquete global
pip install -e .
auto-kubeconfig --help
